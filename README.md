# Lim-IDE
IDE code editor to write programing language code

## Build steps
open terminal in the main project folder.
+ run -> npm install
+ inside the main folder copy lim folder to .\node_modules\codemirror\mode
+ run -> npm start

Write some code :-) .

## System Requirements

+ Mac
+ Linux
+ Windows
